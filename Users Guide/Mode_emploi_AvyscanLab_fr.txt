AVYSCAN LAB — MODE D’EMPLOI COMPLET (FR)
Version document: 23 mars 2026

============================================================
1) OBJECTIF DU LOGICIEL
============================================================
AvyscanLab est une application de restauration de films argentiques numérisés
(8 mm, Super 8, 9,5 mm, 16 mm) basée sur AviSynth+.

Le logiciel permet de:
- prévisualiser en temps réel le rendu du script (lecteur intégré libmpv);
- corriger l’image (crop, grain, poussières, luma/chroma, netteté);
- appliquer des modules personnalisés AviSynth (filtres Custom);
- gérer plusieurs clips dans un même projet;
- encoder en lot avec FFmpeg vers des formats intermédiaires ou finaux.

============================================================
2) PRÉREQUIS ET INSTALLATION
============================================================
Système visé:
- Windows 10/11 64 bits;
- AviSynth+ 64 bits (détection au démarrage);
- FFmpeg et plugins fournis dans l’application.

Démarrage:
1. Lancez AvyscanLab.exe.
2. Si AviSynth+ manque, installez-le puis relancez.
3. Vérifiez l’ouverture correcte du lecteur de preview.

============================================================
3) TOUR DE L’INTERFACE
============================================================
A. Barre de menus
- Infos: Guide utilisateur, aperçu script, visite guidée, mise à jour.
- Settings: threads, source loader, thème (Dark/Grey/Light), accent,
  export/import paramètres, reset paramètres.
- Language: FR, EN, DE, ES.
- Contact / À propos.

B. Zone source et clips
- Onglets multi-clips (ajout +, fermeture x);
- Champ source vidéo/fichier;
- Mode séquence image avec image début/fin;
- FPS et vitesse de lecture.

C. Lecteur
- timeline + temps + frame;
- comparaison Orig / Result;
- navigation image par image;
- mode demi-résolution (preview plus fluide);
- histogramme;
- bouton Record (panneau encodage).

D. Panneau filtres (bas)
- colonne gauche: activation/desactivation modules;
- colonne droite: paramètres du filtre sélectionné;
- gestion des filtres Custom (ajout/import/export/ordre).

============================================================
4) IMPORT SOURCE (VIDÉO OU SÉQUENCE)
============================================================
Méthodes:
- glisser-déposer dans la fenêtre;
- bouton d’ouverture fichier;
- ajout multiple pour traitement en lot.

Formats vidéo principaux:
.avi .mp4 .mov .mkv .wmv .m4v .mpeg .mpg .webm

Formats image:
.tif .tiff .jpg .jpeg .png .bmp

Notes séquences:
- AvyscanLab détecte les noms numériques (ex: 00001.tif) et peut construire
  automatiquement le pattern de séquence.
- renseignez correctement la cadence (fps) selon le format film.

============================================================
5) NAVIGATION ET PREVIEW
============================================================
Raccourcis utiles:
- Espace: lecture/pause
- Flèche gauche/droite: frame précédente/suivante
- Ctrl + gauche/droite: début/fin
- F11: maximiser le viewer

Bonnes pratiques:
- activez 1/2 résolution pour tester rapidement les réglages lourds;
- comparez régulièrement Orig / Result;
- utilisez l’histogramme pour éviter hautes lumières brûlées et noirs bouchés.

============================================================
6) FILTRES NATIFS: RÔLE ET ORDRE CONSEILLÉ
============================================================
Ordre recommandé:
1. Crop
2. Degrain
3. Denoise (RemoveDirt)
4. Luma Levels
5. GamMac
6. Sharpen
7. Custom éventuels

6.1 Crop
Recadre les bords parasites (gauche/haut/droite/bas).

6.2 Degrain (MVTools)
Réduction temporelle du grain.
Paramètres clés:
- mode (mdegrain1/2/3, temporal)
- thSAD / thSADC (intensité)
- blksize / overlap (cohérence importante)
- pel / search / prefilter

6.3 Denoise (RemoveDirt / RemoveDirtMC)
Suppression poussières/rayures.
- mode: removedirtmc recommandé
- strength + dist selon niveau de défaut
- option grey pour agir surtout sur la luma

6.4 Luma Levels
Ajustements:
- luminosité
- contraste
- saturation
- teinte
- gamma luma

6.5 GamMac
Correction colorimétrique avancée:
- LockChan / LockVal
- Scale, Th, HiTh
- zone d’analyse X/Y/W/H
- Omin/Omax, Verbosity

6.6 Sharpen
Netteté finale:
- mode simple ou edge
- strength, radius, threshold

============================================================
7) FILTRES PERSONNALISÉS (CUSTOM)
============================================================
AvyscanLab supporte des modules AviSynth personnalisés avec UI dynamique.

Créer un filtre:
1. Bouton + dans section CUSTOM.
2. Choisissez la position pipeline (avant/après modules natifs).
3. Ajoutez DLL externes si nécessaire.
4. Écrivez le script en utilisant des variables {param}.
5. Définissez le type de contrôle (slider/combo/checkbox/text).
6. Testez via Aperçu puis Enregistrer.

Exploiter un filtre:
- ON/OFF direct dans la liste;
- bouton paramètres pour modifier valeurs;
- édition ultérieure via bouton crayon.

Interopération:
- export/import JSON des filtres custom;
- attention aux chemins DLL absolus sur d’autres machines.

============================================================
8) PRÉRÉGLAGES (PRESETS)
============================================================
Deux niveaux de presets:
A) Presets de restauration (filtres)
- enregistrent votre « recette » de correction.

B) Presets d’encodage
- enregistrent codec/conteneur/qualité/chroma/résolution.

Conseils:
- créez des presets par type de bobine (faible grain, fort grain, etc.);
- nommez clairement (ex: S8_Couleur_FortGrain_v2).

============================================================
9) ENCODAGE ET EXPORT (PANEL RECORD)
============================================================
Flux type:
1. Ouvrez Record.
2. Cochez les clips à encoder (batch list).
3. Définissez dossier de sortie.
4. Choisissez encodeur + conteneur.
5. Définissez qualité (CRF ou bitrate), chroma, résolution.
6. Lancez l’encodage.

Encodeurs proposés (selon build):
- x264 / x265
- FFV1 / UTVideo
- ProRes
- séquences image (TIFF/PNG/JPG)

Fonctions utiles:
- estimation espace disque;
- presets d’encodage;
- arrêt machine après traitement;
- mode essai: durée max par clip (limite trial).

============================================================
10) GESTION MULTI-CLIPS ET SESSION
============================================================
- Chaque onglet conserve ses propres réglages.
- Le nom de preset peut être affiché par clip.
- L’application sauvegarde l’état de session et certains états fenêtre/UI.

Recommandation:
- vérifiez le clip actif avant de modifier des valeurs;
- faites un preset global avant rendu final lot.

============================================================
11) APERÇU SCRIPT AVISYNTH ET DIAGNOSTIC
============================================================
- Menu Infos > Script preview pour consulter le script généré.
- En cas d’erreur de script, utilisez le diagnostic affiché par le lecteur.
- Testez d’abord sans custom filter si un script ne charge plus.

============================================================
15) DÉPANNAGE RAPIDE
============================================================
Problème: preview noire / script ne charge pas
- vérifier chemin source;
- essayer un autre source loader;
- désactiver temporairement custom filters.

Problème: preview lente
- activer 1/2 résolution;
- réduire paramètres lourds (degrain/search/pel);
- augmenter threads progressivement.

Problème: différences de couleurs après export
- vérifier chroma et conteneur;
- comparer en lecture externe avec player de référence.
