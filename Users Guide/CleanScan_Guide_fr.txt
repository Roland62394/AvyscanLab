=============================================================================================
                                  1.  PRESENTATION
=============================================================================================

CleanScan est un logiciel de restauration de films argentiques numerises (8 mm, Super 8,
9,5 mm, 16 mm). Il permet de :

    - Reduire le grain de pellicule (MVTools2 MDegrain)
    - Supprimer poussieres et rayures (RemoveDirt / RemoveDirtMC)
    - Corriger luminosite, contraste, saturation et couleurs
    - Corriger les derives colorimetriques (GamMac)
    - Recadrer l'image
    - Renforcer la nettete (Sharpen / Edge-aware)
    - Ajouter des filtres AviSynth personnalises avec interface de controle

Le logiciel genere un script AviSynth+ optimise et offre une previsualisation en temps reel
via un lecteur video integre (libmpv). L'encodage final est assure par FFmpeg.

L'application est disponible en 4 langues : francais, anglais, allemand, espagnol.


=============================================================================================
                           2.  PREREQUIS ET INSTALLATION
=============================================================================================

Systeme requis :
    - Windows 10 ou 11 (64 bits)
    - 4 Go de RAM minimum (8 Go recommande)
    - Espace disque suffisant pour les fichiers source et les rendus

Composants :
    - AviSynth+ 64 bits doit etre installe. Si absent, CleanScan le detecte au demarrage
      et propose un lien de telechargement. Une version embarquee est utilisee en secours.
    - FFmpeg est fourni avec l'application (dossier Plugins/ffmpeg/).
    - Le lecteur mpv est fourni avec l'application (dossier mpv/).

Aucune installation supplementaire n'est requise. Il suffit de decompresser l'archive
et de lancer l'executable CleanScan.exe.


=============================================================================================
                              3.  INTERFACE GENERALE
=============================================================================================

L'interface est organisee en trois zones principales :

    ZONE SUPERIEURE
    -----------------------------------------------------------------------------------------
    Barre de menus : Infos (guide, apercu script, presets, a propos, contact, reglages),
    Langues (FR/EN/DE/ES), Visite guidee.

    Barre de source : champs de selection du fichier source, mode image/video, threads,
    chargeur source (Auto/FFMS2/L-SMASH/LWLibav).

    Onglets clips : chaque clip charge apparait sous forme d'onglet avec le nom du fichier
    et le preset applique (ex: "[perso2]"). Cliquez sur un onglet pour basculer entre les
    clips. Le bouton + permet d'ajouter un clip, le x ferme un onglet.

    ZONE CENTRALE
    -----------------------------------------------------------------------------------------
    Lecteur video mpv : affiche la previsualisation du script AviSynth en temps reel.

    Barre de transport (en bas du lecteur) :
        |<      Debut du clip
        <<      Image precedente
        >       Lecture / Pause
        >>      Image suivante
        1x      Vitesse (cycle : 0.25x -> 0.5x -> 1x)
        1/2     Demi-resolution (previsualisation plus rapide)
        []      Maximiser/Restaurer le lecteur (F11)
        REC     Ouvrir le panneau d'encodage

    Barre de progression : position actuelle / duree totale.
    Cliquez ou glissez pour naviguer.

    ZONE INFERIEURE
    -----------------------------------------------------------------------------------------
    Panneau de filtres divise en deux colonnes separees par un separateur redimensionnable :
        - Colonne gauche : boutons d'activation des filtres (ON/OFF) et filtres personnalises
        - Colonne droite : parametres detailles du filtre selectionne

    La hauteur entre le lecteur et le panneau de filtres est ajustable via le separateur
    horizontal.


=============================================================================================
                            4.  SELECTION DE LA SOURCE
=============================================================================================

Fichier video
    Glissez-deposez un fichier video sur la zone de lecture, ou cliquez sur le bouton
    a cote du champ "source" pour ouvrir un explorateur de fichiers.
    Formats supportes : .avi  .mp4  .mov  .mkv  .wmv  .webm  .m4v  .flv  .ts  .mts
                        .m2ts  .mxf  .mpeg  .mpg  .ogv  .3gp

Sequence d'images
    Activez le mode image (bouton bascule "IMG"), puis selectionnez un fichier image.
    CleanScan detecte automatiquement la premiere et la derniere image de la sequence.
    Formats supportes : .tif  .tiff  .jpg  .jpeg  .png  .bmp  .gif  .webp  .exr

Cadence de lecture (FPS)
    Le champ "fps" definit la cadence de lecture.
    Valeurs courantes : 16 fps (8 mm), 18 fps (Super 8), 24 fps (16 mm).

Chargeur source
    Le menu deroulant permet de forcer le plugin de decodage :
        Auto      Detection automatique (recommande)
        FFMS2     FFmpegSource2 — compatibilite maximale
        L-SMASH   LSmashSource — MP4/MOV optimise
        LWLibav   LWLibavVideoSource — format alternatif

Threads
    Nombre de threads de traitement AviSynth (defaut : 4). Augmentez pour exploiter
    tous les coeurs du processeur.

Multi-clips
    Vous pouvez charger plusieurs fichiers. Chaque clip dispose de ses propres reglages
    de filtres et de son propre preset. Glissez-deposez plusieurs fichiers a la fois
    ou utilisez le bouton + pour en ajouter.


=============================================================================================
                         5.  LECTEUR VIDEO ET NAVIGATION
=============================================================================================

Le lecteur integre affiche le resultat du script AviSynth en temps reel. Toute modification
de parametre regenere automatiquement le script et rafraichit l'apercu.

Raccourcis clavier :

    Touche                    Action
    -----------------------------------------------------------------------------------------
    Espace                    Lecture / Pause
    Fleche gauche             Image precedente
    Fleche droite             Image suivante
    Ctrl + Gauche             Aller au debut
    Ctrl + Droite             Aller a la fin
    F11                       Maximiser/Restaurer le lecteur

Interactions avec les parametres :

    Action                    Effet
    -----------------------------------------------------------------------------------------
    Clic sur slider           Positionne directement le curseur
    Glisser le slider         Ajustement en temps reel
    Molette sur champ texte   Ajuste la valeur par pas (Step)
    Saisie directe            Tapez une valeur, validez par Entree ou clic ailleurs
    Survol d'un label         Affiche l'infobulle avec plage et conseils

Mode demi-resolution :
    Le bouton "1/2" active le mode demi-resolution : le script AviSynth traite l'image
    a la moitie de sa taille AVANT les filtres lourds, puis l'affiche. Cela accelere
    considerablement la previsualisation sur les machines moins puissantes.

Affichage des erreurs :
    En cas d'erreur de script AviSynth, un bandeau apparait dans le lecteur et une fenetre
    d'erreur s'ouvre avec :
        - Le message d'erreur nettoye (sans les prefixes techniques de ffmpeg)
        - L'extrait du script autour de la ligne fautive (avec marqueur ►)
        - La sortie complete de ffmpeg dans une section "Details" retractable
        - Un bouton "Copier" pour copier le message complet

Preset par clip :
    Le menu deroulant a cote de la barre de transport permet d'appliquer un preset
    de filtres uniquement au clip actif.


=============================================================================================
                              6.  PANNEAU DE FILTRES
=============================================================================================

Le panneau inferieur est divise en deux colonnes :
    Colonne gauche : boutons d'activation (ON/OFF, vert = actif)
    Colonne droite : parametres detailles du filtre selectionne

Chaque filtre possede :
    - Un bouton ON/OFF pour activer/desactiver le filtre
    - Un bouton d'expansion (>) qui affiche son panneau de parametres detailles
    - Des presets rapides (pour certains filtres)

Les options generales (hors filtres) sont :
    preview            Previsualisation AviSynth (comparaison avant/apres)
    enable_flip_h      Retournement horizontal
    enable_flip_v      Retournement vertical

Ordre du pipeline de traitement :
    1. Retournement (flip)
    2. Recadrage (crop)
    3. Suppression poussieres (denoise / RemoveDirt)
    4. Reduction grain (degrain / MVTools2)
    5. Niveaux de luminosite (luma levels)
    6. Correction colorimetrique (GamMac)
    7. Filtres personnalises (6 points d'injection possibles)
    8. Nettete (sharpen) — toujours en dernier

Les valeurs des sliders sont automatiquement alignees sur le pas (step) defini
pour chaque parametre, empechant les valeurs invalides.


=============================================================================================
                           7.  FILTRE CROP (RECADRAGE)
=============================================================================================

Supprime des pixels sur les bords de l'image. Les valeurs sont automatiquement arrondies
aux nombres pairs (exigence du sous-echantillonnage chroma YUV).

    Parametre       Plage          Defaut      Description
    -----------------------------------------------------------------------------------------
    Crop_L          0-500 px       0           Pixels supprimes a gauche
    Crop_T          0-500 px       0           Pixels supprimes en haut
    Crop_R          0-500 px       0           Pixels supprimes a droite
    Crop_B          0-500 px       0           Pixels supprimes en bas


=============================================================================================
                         8.  FILTRES DE RESTAURATION
=============================================================================================

    DEGRAIN — Reduction du grain (MVTools2)
    -----------------------------------------------------------------------------------------
    Reduction temporelle du grain de pellicule via l'analyse de mouvement MVTools2.
    Le filtre compare plusieurs images consécutives pour distinguer le detail du grain.

    Parametre              Plage                        Defaut      Description
    degrain_mode           mdegrain1/2/3/temporal       mdegrain2   Algorithme
    degrain_thSAD          0-1000                       350         Seuil luma
    degrain_thSADC         0-1000                       250         Seuil chroma
    degrain_blksize        4/8/16/32/64                 16          Taille de bloc
    degrain_overlap        0 a blksize/2                blksize/2   Chevauchement
    degrain_pel            1 ou 2                       2           Precision sub-pixel
    degrain_search         0-5                          2           Mode recherche
    degrain_prefilter      remgrain/blur/none           remgrain    Pre-filtre

    Contraintes automatiques :
        - blksize ne peut prendre que les valeurs 4, 8, 16, 32 ou 64 (puissances de 2,
          exigence de MVTools2 MAnalyse). Le slider snape automatiquement.
        - overlap est automatiquement plafonne a blksize/2. Si vous reduisez blksize,
          overlap est reduit en consequence.
        - overlap n'accepte que des valeurs paires (step = 2).

    Presets rapides :
        Grain fin        thSAD=200, thSADC=150
        Grain moyen      thSAD=350, thSADC=250 (defaut)
        Grain fort        thSAD=500, thSADC=400

    Notes :
        - mdegrain2 : 2 references temporelles (recommande, bon rapport qualite/vitesse)
        - mdegrain3 : 3 references (meilleur resultat, plus lent)
        - mdegrain1 : 1 reference (rapide, moins efficace)
        - temporal  : TemporalSoften (sans MVTools2, basique)

    DENOISE — Suppression des poussieres (RemoveDirt)
    -----------------------------------------------------------------------------------------
    Detection et suppression temporelle des poussieres, rayures et defauts ponctuels.

    Parametre              Plage                        Defaut      Description
    denoise_mode           removedirtmc/removedirt      removedirtmc  Algorithme
    denoise_strength       1-24                         10          Seuil de detection
    denoise_dist           1-20                         3           Rayon spatial
    denoise_grey           true/false                   false       Luma uniquement

    Note : removedirtmc (avec compensation de mouvement) est recommande pour de meilleurs
    resultats. Le mode removedirt est plus rapide mais moins precis.


=============================================================================================
                         9.  FILTRES D'AMELIORATION
=============================================================================================

    LUMA LEVELS — Niveaux de luminosite
    -----------------------------------------------------------------------------------------
    Ajustement global de la luminosite, du contraste, de la saturation et de la teinte.

    Parametre       Plage              Defaut      Description
    Lum_Bright      -255 a 255         0           Decalage de luminosite
    Lum_Contrast    0.1 a 3.0          1.05        Multiplicateur de contraste
    Lum_Sat         0.1 a 3.0          1.10        Saturation des couleurs
    Lum_Hue         -180 a 180         0           Rotation de teinte (degres)
    Lum_GammaY      0.1 a 3.0          1.30        Courbe gamma

    Note : GammaY > 1 eclaircit les tons moyens, < 1 les assombrit.
    C'est souvent le parametre le plus utile pour les films anciens sous-exposes.

    GAMMAC — Correction colorimetrique avancee
    -----------------------------------------------------------------------------------------
    Correction automatique des derives de couleur entre canaux RGB. Particulierement
    utile pour les films presentant un jaunissement, une dominante bleue ou rouge.

    Parametre       Plage              Defaut      Description
    LockChan        -3 a 2             1           Canal de reference
    LockVal         1-255              250         Seuil de verrouillage
    Scale           0-2                2           Mode d'echelle
    Th              0.0-1.0            0           Seuil bas
    HiTh            0.0-1.0            1           Seuil haut
    X               0-2000             0           Decalage X de la zone d'analyse
    Y               0-2000             0           Decalage Y de la zone d'analyse
    W               0-4000             0           Largeur de la zone (0 = tout)
    H               0-4000             0           Hauteur de la zone (0 = tout)
    Omin            0-255              0           Sortie minimale
    Omax            0-255              255         Sortie maximale
    Show            true/false         false       Afficher la visualisation
    Verbosity       0-6                0           Niveau de detail du log

    LockChan : 0=Rouge, 1=Vert, 2=Bleu, -1=valeur cible, -2=moyenne, -3=median.
    Cliquez sur le bouton (i) pour ouvrir la documentation detaillee de GamMac.

    SHARPEN — Renforcement de la nettete
    -----------------------------------------------------------------------------------------
    Renforcement de la nettete de l'image. Toujours applique en dernier dans le pipeline
    pour eviter d'amplifier les artefacts des autres filtres.

    Parametre              Plage              Defaut      Description
    Sharp_Mode             simple/edge        simple      Algorithme
    Sharp_Strength         1-20               8           Force du renforcement
    Sharp_Radius           0.5-5.0            1.5         Rayon de flou (masque)
    Sharp_Threshold        0-100              0           Seuil anti-bruit

    Presets rapides :
        Leger            Strength=5
        Standard         Strength=8 (defaut)
        Moyen            Strength=12
        Fort             Strength=16
        Tres fort        Strength=20

    Notes :
        - simple : Sharpen() global, efficace mais peut amplifier le grain residuel.
        - edge : contours uniquement via masque Sobel + gamma (technique
          LimitedSharpenFaster). Preserve les zones lisses. Recommande apres Degrain.


=============================================================================================
                        10.  FILTRES PERSONNALISES (CUSTOM)
=============================================================================================

CleanScan permet de creer vos propres filtres AviSynth et de les integrer dans le pipeline
de restauration avec une interface de controle complete.

    CREER UN FILTRE PERSONNALISE
    -----------------------------------------------------------------------------------------

    1. Cliquez sur le bouton "+" a droite du titre "CUSTOM" dans le panneau de filtres.
       Le formulaire de filtre personnalise s'ouvre.

    2. Renseignez le nom du filtre et la position dans le pipeline :
       - BeforePipeline : avant tous les filtres
       - AfterGamMac    : apres la correction colorimetrique
       - AfterDenoise   : apres la suppression des poussieres
       - AfterDegrain   : apres la reduction du grain
       - AfterLuma      : apres les niveaux de luminosite
       - AfterSharpen   : apres la nettete (fin du pipeline)

    3. Si votre filtre necessite un plugin AviSynth externe (.dll), cliquez sur
       "+ Ajouter DLL..." pour l'ajouter. Les DLLs sont chargees via TryLoadPlugin()
       et les doublons sont automatiquement elimines.

    4. Ecrivez le code AviSynth dans l'editeur. Utilisez "c" comme variable clip :
           c = MonFiltre(c, force={sigma})
       Les mots entre accolades {nom} sont detectes automatiquement comme parametres.

    5. Pour chaque parametre detecte, configurez le type de controle :
       - slider   : curseur avec Min, Max, Step et Defaut.
                     Les valeurs sont automatiquement alignees sur le pas (step).
                     Avec un step de 2, seules les valeurs paires sont autorisees.
       - combo    : liste deroulante avec valeurs separees par virgule
       - checkbox : case a cocher avec valeurs On/Off personnalisables
       - text     : champ texte libre

    6. Utilisez le bouton "Apercu" pour tester le filtre en temps reel sans quitter
       le formulaire.

    7. Cliquez sur "Enregistrer" pour sauvegarder.

    UTILISER UN FILTRE PERSONNALISE
    -----------------------------------------------------------------------------------------

    Le filtre apparait dans la section CUSTOM avec :
        - Un bouton ON/OFF pour activer/desactiver le filtre
        - Un bouton fleche (>) pour ouvrir le panneau de parametres
        - Un bouton crayon pour ouvrir le formulaire d'edition

    Les parametres sont affiches dans la colonne droite du panneau de filtres avec les
    controles definis (sliders, listes, cases a cocher, champs texte).

    EXPORTER / IMPORTER UN FILTRE
    -----------------------------------------------------------------------------------------

    Exporter : dans le formulaire d'edition, cliquez sur "Exporter". Le filtre est
    sauvegarde dans un fichier .json que vous pouvez partager avec d'autres utilisateurs.

    Importer : dans le panneau de filtres, cliquez sur le bouton fleche vers le bas a cote
    du "+" dans la section CUSTOM. Selectionnez un fichier .json exporte. Les filtres
    importes recoivent un nouvel identifiant pour eviter les doublons.

    Note : les chemins DLL sont absolus. Si le filtre utilise un plugin externe, assurez-vous
    que la DLL est presente au meme emplacement sur la machine cible, ou editez le filtre
    apres importation pour corriger le chemin.

    SUPPRIMER UN FILTRE
    -----------------------------------------------------------------------------------------

    Ouvrez le formulaire d'edition (bouton crayon) et cliquez sur "Supprimer".

Emplacement de stockage : %AppData%\CleanScan\custom_filters.json


=============================================================================================
                            11.  PREREGLAGES (PRESETS)
=============================================================================================

    PRESETS GLOBAUX (filtres)
    -----------------------------------------------------------------------------------------
    Le menu Infos > Preset permet de sauvegarder et restaurer vos configurations de filtres.

    Action              Description
    Sauvegarder         Enregistre les reglages actuels sous un nom
    Charger             Applique un preset existant (s'applique a tous les clips charges)
    Supprimer           Efface un preset

    Les champs source, film, img, start, end et crop ne sont PAS inclus dans les presets.
    Emplacement : %AppData%\CleanScan\presets.json

    PRESETS PAR CLIP
    -----------------------------------------------------------------------------------------
    Chaque clip peut avoir ses propres reglages de filtres. Quand vous modifiez manuellement
    un parametre, le clip est automatiquement nomme "perso1", "perso2", etc.
    Le menu deroulant dans la barre de transport permet de charger un preset global pour
    le clip actif uniquement.

    PRESETS D'ENCODAGE
    -----------------------------------------------------------------------------------------
    Les combinaisons d'encodage (encodeur + conteneur + qualite + chroma + resolution)
    peuvent etre sauvegardees et rechargees depuis le panneau Record.
    Le preset "Default" est toujours present et apparait en premier.
    Emplacement : %AppData%\CleanScan\encoding_presets.json


=============================================================================================
                              12.  ENCODAGE ET RENDU
=============================================================================================

Cliquez sur le bouton REC (rouge) dans la barre de transport pour ouvrir le panneau
d'encodage.

    ETAPE 1 — Selection des clips
    -----------------------------------------------------------------------------------------
    Cochez les clips a encoder dans la liste. Utilisez "Tout selectionner" pour les
    encoder tous en une seule fois (batch). Chaque clip peut avoir un nom de sortie
    et un preset d'encodage differents.

    ETAPE 2 — Dossier de destination
    -----------------------------------------------------------------------------------------
    Cliquez sur le bouton parcourir ou saisissez le chemin. L'espace disque disponible
    est affiche en temps reel.

    ETAPE 3 — Choix de l'encodeur
    -----------------------------------------------------------------------------------------
    Encodeur       Type             Usage
    x264           Lossy            Diffusion (H.264)
    x265           Lossy            Diffusion (HEVC)
    FFV1           Lossless         Archivage professionnel
    UT Video       Lossless         Edition rapide
    ProRes         Lossy HQ         Post-production
    TIFF seq.      Lossless         Images individuelles (archivage)
    PNG seq.       Lossless         Images individuelles
    JPG seq.       Lossy            Images individuelles

    ETAPE 4 — Conteneur
    -----------------------------------------------------------------------------------------
    MKV, MP4, AVI ou MOV (pour les codecs video). Ignore pour les sequences d'images.

    ETAPE 5 — Parametres avances (codecs lossy uniquement)
    -----------------------------------------------------------------------------------------
    Mode qualite :
        CRF      Qualite constante (defaut). Plus la valeur est basse, meilleure est la
                 qualite. Valeurs typiques : 18-23. Le slider va de 0 a 51.
        Bitrate  Debit fixe en Mb/s. Le minimum recommande est calcule automatiquement
                 en fonction de la resolution et du chroma choisis.

    Sous-echantillonnage chroma :
        4:2:0    Standard (plus petit fichier)
        4:2:2    Meilleure qualite couleur
        4:4:4    Qualite maximale (fichier volumineux)

        Note : le bitrate et le chroma sont couples. Modifier le chroma ajuste le
        bitrate minimum recommande. Entrer un bitrate eleve peut ajuster
        automatiquement le chroma au niveau maximum supporte.

    Resolution de sortie :
        Original   Resolution source conservee
        1080p      1920x1080
        720p       1280x720
        576p       720x576 (PAL)
        480p       720x480 (NTSC)

        Note : les dimensions impaires sont automatiquement ajustees aux nombres pairs
        (exigence de x264/x265) via un filtre de padding.

    PENDANT L'ENCODAGE
    -----------------------------------------------------------------------------------------
    - La lecture video est automatiquement arretee
    - L'interface est verrouillee (filtres, transport, menus, onglets clips)
    - Seul le bouton "Arreter" reste actif
    - Une barre de progression affiche le pourcentage d'avancement
    - La fermeture de l'application stoppe automatiquement l'encodage en cours

    Option "Eteindre apres encodage" : cochez cette case pour arreter l'ordinateur
    automatiquement apres un encodage batch reussi.

    Note technique : l'encodage utilise un script de rendu dedie (ScriptRender.avs) ou
    le mode preview est desactive, garantissant la qualite finale maximale.

=============================================================================================
                             16.  WORKFLOW RECOMMANDE
=============================================================================================

    1. Ouvrez votre source (glisser-deposer ou champ source).

    2. Naviguez jusqu'a une image representative (ni trop sombre, ni trop claire).

    3. Activez les filtres un par un dans l'ordre du pipeline :

       a) Crop       — Supprimez les bordures noires ou ajustez le cadrage.

       b) Denoise    — Supprimez les poussieres et rayures. Le preset "standard"
                       convient dans la plupart des cas. Augmentez strength si les
                       defauts persistent.

       c) Degrain    — Commencez par le preset "Grain moyen", puis augmentez thSAD
                       si le grain persiste. Attention : des valeurs trop elevees
                       produisent un effet "plastique".

       d) Luma       — Ajustez luminosite et contraste. GammaY est souvent le
                       parametre le plus utile pour eclaircir les films anciens.

       e) GamMac     — Corrigez les derives de couleur (jaunissement, dominante
                       bleue, rouge, etc.).

       f) Sharpen    — Renforcez la nettete en derniere etape. Le mode "edge"
                       preserve mieux les zones lisses et evite d'amplifier le
                       grain residuel.

    4. Utilisez les presets de chaque filtre pour un reglage rapide, puis affinez.

    5. Verifiez le resultat en parcourant plusieurs passages du film.

    6. Sauvegardez votre configuration en preset.

    7. Si plusieurs clips : chargez-les tous, appliquez les presets adaptes.

    8. Ouvrez le panneau Record, choisissez l'encodeur et lancez l'encodage final.

    Conseil : activez le mode demi-resolution (1/2) pendant le reglage des parametres
    pour accelerer la previsualisation, puis desactivez-le pour verifier le resultat
    en pleine resolution avant l'encodage.

                           17.  ASTUCES ET RACCOURCIS
=============================================================================================

    - La molette de souris sur les champs numeriques ajuste la valeur par pas.
    - Le mode demi-resolution (bouton 1/2) accelere considerablement la previsualisation.
    - Tous les parametres sont sauvegardes automatiquement dans ScriptUser.avs a chaque
      modification.
    - La position, la taille de la fenetre et l'etat des panneaux sont memorises entre
      les sessions.
    - Le panneau Record est restaure a l'ouverture s'il etait ouvert a la fermeture.
    - Les infobulles (survol) sur chaque parametre affichent les plages et des conseils.
    - Le glisser-deposer de plusieurs fichiers cree automatiquement un clip par fichier.
    - Les sliders se positionnent directement au clic (pas besoin de glisser).
    - La visite guidee (menu) permet de decouvrir l'interface pas a pas.
    - Les erreurs AviSynth affichent automatiquement la ligne de script fautive.
    - Le bouton "Copier" dans la fenetre d'erreur copie le message + les details.
