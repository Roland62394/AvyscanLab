=============================================================================================
                                  1.  PRÉSENTATION
=============================================================================================

CleanScan est un logiciel de restauration de films argentiques numérisés (8 mm, Super 8,
9,5 mm, 16 mm). Il permet de :

    - Réduire le grain de pellicule (MVTools2 MDegrain)
    - Supprimer les poussières et les rayures (RemoveDirt / RemoveDirtMC)
    - Corriger la luminosité, le contraste, la saturation et les couleurs
    - Corriger les dérives colorimétriques (GamMac)
    - Recadrer l’image
    - Renforcer la netteté (Sharpen / Edge-aware)
    - Ajouter des filtres AviSynth personnalisés avec interface de contrôle

Le logiciel génère un script AviSynth+ optimisé et offre une prévisualisation en temps réel
via un lecteur vidéo intégré (libmpv). L’encodage final est assuré par FFmpeg.

L’application est disponible en 4 langues : français, anglais, allemand et espagnol.

=============================================================================================
                                REMERCIEMENTS / CRÉDITS
=============================================================================================

CleanScan s’appuie sur la qualité des plugins AviSynth+ développés
et maintenus par la communauté open source. Nous adressons nos remerciements les plus chaleureux,
aussi exhaustivement que possible, à leurs auteurs :

     MVTools2 — Manao, Fizick, Tsp, TSchniede, SEt
          Estimation de mouvement et réduction temporelle du grain (MDegrain).
          Le cœur du pipeline de restauration de CleanScan.

      RemoveDirt — Kassandro
          Détection et suppression temporelles des poussières et des rayures.
          RemoveDirtMC (avec compensation de mouvement) reste inégalé pour
          le nettoyage des films argentiques.

      GamMac — StainlessS
          Correction colorimétrique avancée par analyse des canaux RVB.
          Un outil précieux pour corriger les dérives de couleur
          des pellicules vieillissantes.

      MaskTools2 — Manao, tp7
          Opérations sur les masques, indispensables pour le renforcement
          sélectif de la netteté (mode edge-aware).

      RgTools — tp7, pinterf
          Outils de suppression du grain et opérations matricielles,
          utilisés comme préfiltre dans l’analyse de mouvement.

      LimitedSharpenFaster — Didée
          Technique de renforcement de la netteté par masque Sobel et courbe
          gamma (^0.86), dont s’inspire le mode « edge » de CleanScan.

=============================================================================================
                           2.  PRÉREQUIS ET INSTALLATION
=============================================================================================

Système requis :
    - Windows 10 ou 11 (64 bits)
    - 4 Go de RAM minimum (8 Go recommandés)
    - Espace disque suffisant pour les fichiers source et les rendus

Composants :
    - AviSynth+ 64 bits doit être installé. S’il est absent, CleanScan le détecte au démarrage
      et propose son installation.
    - FFmpeg est fourni avec l’application (dossier Plugins/ffmpeg/).
    - Le lecteur mpv est fourni avec l’application (dossier mpv/).

Aucune installation supplémentaire n’est requise. Il suffit de décompresser l’archive
et de lancer l’exécutable CleanScan.exe.


=============================================================================================
                              3.  INTERFACE GÉNÉRALE
=============================================================================================

L’interface est organisée en trois zones principales :

    ZONE SUPÉRIEURE
    -----------------------------------------------------------------------------------------
    Barre de menus : Infos (guide, aperçu du script, presets, à propos, contact, réglages),
    Langues (FR/EN/DE/ES), visite guidée.

    Barre de source : champs de sélection du fichier source, mode image/vidéo, threads,
    chargeur source (Auto/FFMS2/L-SMASH/LWLibav).

    Onglets clips : chaque clip chargé apparaît sous forme d’onglet avec le nom du fichier
    et le preset appliqué (ex. : "[perso2]"). Cliquez sur un onglet pour basculer entre les
    clips. Le bouton + permet d’ajouter un clip, le x ferme un onglet.

    ZONE CENTRALE
    -----------------------------------------------------------------------------------------
    Lecteur vidéo mpv : affiche la prévisualisation du script AviSynth en temps réel.

    Barre de transport (en bas du lecteur) :
        |<      Début du clip
        <<      Image précédente
        >       Lecture / Pause
        >>      Image suivante
        1x      Vitesse (cycle : 0.25x -> 0.5x -> 1x)
        1/2     Demi-résolution (prévisualisation plus rapide)
        []      Maximiser/Restaurer le lecteur (F11)
        REC     Ouvrir le panneau d’encodage

    Barre de progression : position actuelle / durée totale.
    Cliquez ou glissez pour naviguer.

    ZONE INFÉRIEURE
    -----------------------------------------------------------------------------------------
    Panneau de filtres divisé en deux colonnes séparées par un séparateur redimensionnable :
        - Colonne gauche : boutons d’activation des filtres (ON/OFF) et filtres personnalisés
        - Colonne droite : paramètres détaillés du filtre sélectionné

    La hauteur entre le lecteur et le panneau de filtres est ajustable via le séparateur
    horizontal.


=============================================================================================
                            4.  SÉLECTION DE LA SOURCE
=============================================================================================

Fichier vidéo
    Glissez-déposez un fichier vidéo sur la zone de lecture, ou cliquez sur le bouton
    à côté du champ "source" pour ouvrir un explorateur de fichiers.
    Formats supportés : .avi  .mp4  .mov  .mkv  .wmv  .webm  .m4v  .flv  .ts  .mts
                        .m2ts  .mxf  .mpeg  .mpg  .ogv  .3gp

Séquence d’images
    Sélectionnez un répertoire contenant des fichiers images (par exemple : 00001.tif, 00002.tif, etc.).
    CleanScan détecte automatiquement la première et la dernière image de la séquence.
    Formats supportés : .tif  .tiff  .jpg  .jpeg  .png  .bmp

Cadence de lecture (FPS)
    Le champ "fps" définit la cadence de lecture.
    Valeurs courantes pour les formats usuels :
                    8 mm : 16 ou 18 ips
                    Super 8 : 18 ou 24 ips
                    16 mm : 18 ou 24 ips

Chargeur source
    Le menu déroulant permet de forcer le plugin de décodage :
        Auto      Détection automatique (recommandée)
        FFMS2     FFmpegSource2 — compatibilité maximale
        L-SMASH   LSmashSource — MP4/MOV optimisé
        LWLibav   LWLibavVideoSource — format alternatif
    En cas d’erreur de lecture à l’ouverture d’un fichier, essayez de changer le chargeur source.

Threads
    Nombre de threads de traitement AviSynth (défaut : 4). Augmentez cette valeur pour exploiter
    tous les cœurs du processeur.
    Une valeur trop élevée peut provoquer des ralentissements ou des erreurs sur les machines les moins puissantes.
    Ajustez-la en fonction de votre matériel.

Multi-clips
    Vous pouvez charger plusieurs fichiers. Chaque clip dispose de ses propres réglages
    de filtres et de son propre preset. Glissez-déposez plusieurs fichiers à la fois
    ou utilisez le bouton + pour en ajouter.


=============================================================================================
                         5.  LECTEUR VIDÉO ET NAVIGATION
=============================================================================================

Le lecteur intégré affiche le résultat du script AviSynth en temps réel. Toute modification
de paramètre régénère automatiquement le script et rafraîchit l’aperçu.

Raccourcis clavier :

    Touche                    Action
    -----------------------------------------------------------------------------------------
    Espace                    Lecture / Pause
    Flèche gauche             Image précédente
    Flèche droite             Image suivante
    Ctrl + Gauche             Aller au début
    Ctrl + Droite             Aller à la fin
    F11                       Maximiser/Restaurer la taille du lecteur

Interactions avec les paramètres :

    Action                    Effet
    -----------------------------------------------------------------------------------------
    Clic sur slider           Positionne directement le curseur
    Glisser le slider         Ajustement en temps réel
    Molette sur champ texte   Ajuste la valeur par pas (step)
    Saisie directe            Tapez une valeur, validez par Entrée ou clic ailleurs
    Survol d’un label         Affiche l’infobulle avec plage et conseils liés au paramètre sélectionné

Mode demi-résolution :
    Le bouton "1/2" active le mode demi-résolution : le script AviSynth traite l’image
    à la moitié de sa taille AVANT les filtres lourds, puis l’affiche. Cela accélère
    considérablement la prévisualisation sur les machines moins puissantes.

Affichage des erreurs :
    En cas d’erreur de script AviSynth, un bandeau apparaît dans le lecteur et une fenêtre
    d’erreur s’ouvre avec :
        - Le message d’erreur (sans les préfixes techniques de ffmpeg)
        - L’extrait du script autour de la ligne fautive (avec marqueur ►)
        - La sortie complète de ffmpeg dans une section "Détails" rétractable
        - Un bouton "Copier" pour copier le message complet

Preset par clip :
    Le menu déroulant à côté de la barre de transport permet d’appliquer un preset
    de filtres uniquement au clip actif.


=============================================================================================
                              6.  PANNEAU DE FILTRES
=============================================================================================

Le panneau inférieur est divisé en deux colonnes :
    Colonne gauche : boutons d’activation (ON/OFF, vert = actif)
    Colonne droite : paramètres détaillés du filtre sélectionné

Chaque filtre possède :
    - Un bouton ON/OFF pour activer/désactiver le filtre
    - Un bouton d’expansion (>) qui affiche son panneau avec ses paramètres détaillés
    - Des presets rapides (pour certains filtres)

Les options générales (hors filtres) sont :
    preview            Prévisualisation AviSynth (comparaison avant/après)
    enable_flip_h      Retournement horizontal
    enable_flip_v      Retournement vertical

Ordre du pipeline de traitement :
    1. Retournement (flip)
    2. Recadrage (crop)
    3. Suppression des poussières (denoise / RemoveDirt)
    4. Réduction du grain (degrain / MVTools2)
    5. Niveaux de luminosité (luma levels)
    6. Correction colorimétrique (GamMac)
    7. Filtres personnalisés (6 points d’injection possibles)
    8. Netteté (sharpen) — toujours en dernier

Les valeurs des sliders sont automatiquement alignées sur le pas (step) défini
pour chaque paramètre, ce qui empêche les valeurs invalides.


=============================================================================================
                           7.  FILTRE CROP (RECADRAGE)
=============================================================================================

Supprime des pixels sur les bords de l’image. Les valeurs sont automatiquement arrondies
aux nombres pairs (exigence du sous-échantillonnage chroma YUV).

    Paramètre       Plage          Défaut      Description
    -----------------------------------------------------------------------------------------
    Crop_L          0-500 px       0           Pixels supprimés à gauche
    Crop_T          0-500 px       0           Pixels supprimés en haut
    Crop_R          0-500 px       0           Pixels supprimés à droite
    Crop_B          0-500 px       0           Pixels supprimés en bas


=============================================================================================
                         8.  FILTRES DE RESTAURATION
=============================================================================================

    DEGRAIN — Réduction du grain (MVTools2)
    -----------------------------------------------------------------------------------------
    Réduction temporelle du grain de pellicule via l’analyse de mouvement MVTools2.
    Le filtre compare plusieurs images consécutives pour distinguer le détail du grain.

    Paramètre              Plage                        Défaut      Description
    degrain_mode           mdegrain1/2/3/temporal       mdegrain2   Algorithme
    degrain_thSAD          0-1000                       350         Seuil luma
    degrain_thSADC         0-1000                       250         Seuil chroma
    degrain_blksize        4/8/16/32/64                 16          Taille de bloc
    degrain_overlap        0 à blksize/2                blksize/2   Chevauchement
    degrain_pel            1 ou 2                       2           Précision sub-pixel
    degrain_search         0-5                          2           Mode de recherche
    degrain_prefilter      remgrain/blur/none           remgrain    Préfiltre

    Contraintes automatiques :
        - blksize ne peut prendre que les valeurs 4, 8, 16, 32 ou 64 (puissances de 2,
          exigence de MVTools2 MAnalyse). Le slider s’ajuste automatiquement.
        - overlap est automatiquement plafonné à blksize/2. Si vous réduisez blksize,
          overlap est réduit en conséquence.
        - overlap n’accepte que des valeurs paires (step = 2).

    Presets rapides :
        Grain fin        thSAD=200, thSADC=150
        Grain moyen      thSAD=350, thSADC=250 (défaut)
        Grain fort       thSAD=500, thSADC=400

    Notes :
        - mdegrain2 : 2 références temporelles (recommandé, bon rapport qualité/vitesse)
        - mdegrain3 : 3 références (meilleur résultat, plus lent)
        - mdegrain1 : 1 référence (rapide, moins efficace)
        - temporal  : TemporalSoften (sans MVTools2, basique)

    DENOISE — Suppression des poussières (RemoveDirt)
    -----------------------------------------------------------------------------------------
    Détection et suppression temporelles des poussières et défauts ponctuels.

    Paramètre              Plage                        Défaut         Description
    denoise_mode           removedirtmc/removedirt      removedirtmc   Algorithme
    denoise_strength       1-24                         10             Seuil de détection
    denoise_dist           1-20                         3              Rayon spatial
    denoise_grey           true/false                   false          Luma uniquement

    Note : removedirtmc (avec compensation de mouvement) est recommandé pour de meilleurs
    résultats. Le mode removedirt est plus rapide mais moins précis.


=============================================================================================
                         9.  FILTRES D’AMÉLIORATION
=============================================================================================

    LUMA LEVELS — Niveaux de luminosité
    -----------------------------------------------------------------------------------------
    Ajustement global de la luminosité, du contraste, de la saturation et de la teinte.

    Paramètre       Plage              Défaut      Description
    Lum_Bright      -255 à 255         0           Décalage de luminosité
    Lum_Contrast    0.1 à 3.0          1.05        Multiplicateur de contraste
    Lum_Sat         0.1 à 3.0          1.10        Saturation des couleurs
    Lum_Hue         -180 à 180         0           Rotation de teinte (degrés)
    Lum_GammaY      0.1 à 3.0          1.30        Courbe gamma

    Note : GammaY > 1 éclaircit les tons moyens, < 1 les assombrit.
    C’est souvent le paramètre le plus utile pour les films anciens sous-exposés.

    GAMMAC — Correction colorimétrique avancée
    -----------------------------------------------------------------------------------------
    Correction automatique des dérives de couleur entre canaux RGB. Particulièrement
    utile pour les films présentant un jaunissement, une dominante bleue ou rouge.

    Paramètre       Plage              Défaut      Description
    LockChan        -3 à 2             1           Canal de référence
    LockVal         1-255              250         Seuil de verrouillage
    Scale           0-2                2           Mode d’échelle
    Th              0.0-1.0            0           Seuil bas
    HiTh            0.0-1.0            1           Seuil haut
    X               0-2000             0           Décalage X de la zone d’analyse
    Y               0-2000             0           Décalage Y de la zone d’analyse
    W               0-4000             0           Largeur de la zone (0 = tout)
    H               0-4000             0           Hauteur de la zone (0 = tout)
    Omin            0-255              0           Sortie minimale
    Omax            0-255              255         Sortie maximale
    Show            true/false         false       Afficher la visualisation
    Verbosity       0-6                0           Niveau de détail du log

    LockChan : 0=Rouge, 1=Vert, 2=Bleu, -1=valeur cible, -2=moyenne, -3=médiane.
    Cliquez sur le bouton (i) pour ouvrir la documentation détaillée de GamMac.

    SHARPEN — Renforcement de la netteté
    -----------------------------------------------------------------------------------------
    Renforcement de la netteté de l’image. Toujours appliqué en dernier dans le pipeline
    pour éviter d’amplifier les artefacts des autres filtres.

    Paramètre              Plage              Défaut      Description
    Sharp_Mode             simple/edge        simple      Algorithme
    Sharp_Strength         1-20               8           Force du renforcement
    Sharp_Radius           0.5-5.0            1.5         Rayon de flou (masque)
    Sharp_Threshold        0-100              0           Seuil anti-bruit

    Presets rapides :
        Léger            Strength=5
        Standard         Strength=8 (défaut)
        Moyen            Strength=12
        Fort             Strength=16
        Très fort        Strength=20

    Notes :
        - simple : Sharpen() global, efficace mais peut amplifier le grain résiduel.
        - edge : contours uniquement via masque Sobel + gamma (technique
          LimitedSharpenFaster). Préserve les zones lisses. Recommandé après Degrain.


=============================================================================================
                        10.  FILTRES PERSONNALISÉS (CUSTOM)
=============================================================================================

CleanScan permet de créer vos propres filtres AviSynth et de les intégrer dans le pipeline
de restauration avec une interface de contrôle complète.

    CRÉER UN FILTRE PERSONNALISÉ
    -----------------------------------------------------------------------------------------

    1. Cliquez sur le bouton "+" à droite du titre "CUSTOM" dans le panneau de filtres.
       Le formulaire de filtre personnalisé s’ouvre.

    2. Renseignez le nom du filtre et la position dans le pipeline :
       - BeforePipeline : avant tous les filtres
       - AfterGamMac    : après la correction colorimétrique
       - AfterDenoise   : après la suppression des poussières
       - AfterDegrain   : après la réduction du grain
       - AfterLuma      : après les niveaux de luminosité
       - AfterSharpen   : après la netteté (fin du pipeline)

    3. Si votre filtre nécessite un plugin AviSynth externe (.dll), cliquez sur
       "+ Ajouter DLL..." pour l’ajouter. Les DLL sont chargées via TryLoadPlugin()
       et les doublons sont automatiquement éliminés.

    4. Écrivez le code AviSynth dans l’éditeur. Utilisez "c" comme variable clip :
           c = MonFiltre(c, force={sigma})
       Les mots entre accolades {nom} sont détectés automatiquement comme paramètres.

    5. Pour chaque paramètre détecté, configurez le type de contrôle :
       - slider   : curseur avec Min, Max, Step et Défaut.
                    Les valeurs sont automatiquement alignées sur le pas (step).
                    Avec un step de 2, seules les valeurs paires sont autorisées.
       - combo    : liste déroulante avec valeurs séparées par des virgules
       - checkbox : case à cocher avec valeurs On/Off personnalisables
       - text     : champ texte libre

    6. Utilisez le bouton "Aperçu" pour tester le filtre en temps réel sans quitter
       le formulaire.

    7. Cliquez sur "Enregistrer" pour sauvegarder.

    UTILISER UN FILTRE PERSONNALISÉ
    -----------------------------------------------------------------------------------------

    Le filtre apparaît dans la section CUSTOM avec :
        - Un bouton ON/OFF pour activer/désactiver le filtre
        - Un bouton flèche (>) pour ouvrir le panneau de paramètres
        - Un bouton crayon pour ouvrir le formulaire d’édition

    Les paramètres sont affichés dans la colonne droite du panneau de filtres avec les
    contrôles définis (sliders, listes, cases à cocher, champs texte).

    EXPORTER / IMPORTER UN FILTRE
    -----------------------------------------------------------------------------------------

    Exporter : dans le formulaire d’édition, cliquez sur "Exporter". Le filtre est
    sauvegardé dans un fichier .json que vous pouvez partager avec d’autres utilisateurs.

    Importer : dans le panneau de filtres, cliquez sur le bouton flèche vers le bas à côté
    du "+" dans la section CUSTOM. Sélectionnez un fichier .json exporté. Les filtres
    importés reçoivent un nouvel identifiant pour éviter les doublons.

    Note : les chemins DLL sont absolus. Si le filtre utilise un plugin externe, assurez-vous
    que la DLL est présente au même emplacement sur la machine cible, ou éditez le filtre
    après importation pour corriger le chemin.

    SUPPRIMER UN FILTRE
    -----------------------------------------------------------------------------------------

    Ouvrez le formulaire d’édition (bouton crayon) et cliquez sur "Supprimer".

Emplacement de stockage : %AppData%\CleanScan\custom_filters.json


  IMPORTER UN SCRIPT AVSI
      -----------------------------------------------------------------------------------------

      Certains filtres AviSynth reposent sur des fonctions définies dans des fichiers
      de script (.avsi). Ces fichiers contiennent des fonctions réutilisables qui étendent
      les possibilités d’AviSynth sans être des plugins compilés (.dll).

      Pour ajouter un script .avsi à un filtre personnalisé :

      1. Ouvrez le formulaire d’édition du filtre (bouton crayon ou création d’un nouveau
         filtre via le bouton "+").

      2. Dans la section "Scripts (Import .avsi)", cliquez sur "+ Ajouter AVSI...".

      3. Sélectionnez un ou plusieurs fichiers .avsi ou .avs depuis votre disque.
         Les fichiers sélectionnés sont automatiquement copiés dans le dossier
         Plugins/Scripts/ de l’application, ce qui garantit leur disponibilité
         indépendamment de leur emplacement d’origine.

      4. Les scripts ajoutés apparaissent dans la liste. Utilisez le bouton (x) pour
         retirer un script de la liste.

      5. Lors de la génération du script AviSynth, CleanScan insère automatiquement
         une instruction Import() pour chaque script référencé, juste après le
         chargement des plugins :

             TryLoadPlugin("C:/.../MonPlugin.dll")
             Import("C:/.../Plugins/Scripts/MesFonctions.avsi")

      Les fonctions définies dans le fichier .avsi sont alors disponibles dans le code
      de votre filtre personnalisé.

      Exemple : si votre fichier MesFonctions.avsi définit une fonction MonDebruitage(),
      vous pouvez l’utiliser directement dans le code du filtre :
          c = MonDebruitage(c, force={sigma})

      Note : lors de l’exportation d’un filtre, seul le nom du fichier .avsi est
      enregistré (pas son contenu). Si vous partagez un filtre avec un autre utilisateur,
      transmettez également les fichiers .avsi nécessaires. L’utilisateur devra les
      importer via le même mécanisme.



=============================================================================================
                            11.  PRÉRÉGLAGES (PRESETS)
=============================================================================================

    PRESETS GLOBAUX (filtres)
    -----------------------------------------------------------------------------------------
    Le menu Infos > Presets permet de sauvegarder et restaurer vos configurations de filtres.

    Action              Description
    Sauvegarder         Enregistre les réglages actuels sous un nom
    Charger             Applique un preset existant (s’applique à tous les clips chargés)
    Supprimer           Efface un preset

    Les champs source, film, img, start, end et crop ne sont PAS inclus dans les presets.
    Emplacement : %AppData%\CleanScan\presets.json

    PRESETS PAR CLIP
    -----------------------------------------------------------------------------------------
    Chaque clip peut avoir ses propres réglages de filtres. Quand vous modifiez manuellement
    un paramètre, le clip est automatiquement nommé "perso1", "perso2", etc.
    Le menu déroulant dans la barre de transport permet de charger un preset global pour
    le clip actif uniquement.

    PRESETS D’ENCODAGE
    -----------------------------------------------------------------------------------------
    Les combinaisons d’encodage (encodeur + conteneur + qualité + chroma + résolution)
    peuvent être sauvegardées et rechargées depuis le panneau Record.
    Le preset "Default" est toujours présent et apparaît en premier.
    Emplacement : %AppData%\CleanScan\encoding_presets.json


=============================================================================================
                              12.  ENCODAGE ET RENDU
=============================================================================================

Cliquez sur le bouton REC (rouge) dans la barre de transport pour ouvrir le panneau
d’encodage.

    ÉTAPE 1 — Sélection des clips
    -----------------------------------------------------------------------------------------
    Cochez les clips à encoder dans la liste. Utilisez "Tout sélectionner" pour les
    encoder tous en une seule fois (batch). Chaque clip peut avoir un nom de sortie
    et un preset d’encodage différents.
    Chaque preset peut inclure un dossier de destination différent, ce qui permet d’organiser
    facilement les rendus de plusieurs clips.

    ÉTAPE 2 — Dossier de destination
    -----------------------------------------------------------------------------------------
    Cliquez sur le bouton Parcourir ou saisissez le chemin. L’espace disque disponible
    est affiché en temps réel.

    ÉTAPE 3 — Choix de l’encodeur
    -----------------------------------------------------------------------------------------
    Encodeur       Type             Usage
    x264           Lossy            Diffusion (H.264)
    x265           Lossy            Diffusion (HEVC)
    FFV1           Lossless         Archivage professionnel
    UT Video       Lossless         Édition rapide
    ProRes         Lossy HQ         Post-production
    TIFF seq.      Lossless         Images individuelles (archivage)
    PNG seq.       Lossless         Images individuelles
    JPG seq.       Lossy            Images individuelles

    ÉTAPE 4 — Conteneur
    -----------------------------------------------------------------------------------------
    MKV, MP4, AVI ou MOV (pour les codecs vidéo). Ignoré pour les séquences d’images.

    ÉTAPE 5 — Paramètres avancés (codecs lossy uniquement)
    -----------------------------------------------------------------------------------------
    Mode qualité :
        CRF      Qualité constante (défaut). Plus la valeur est basse, meilleure est la
                 qualité. Valeurs typiques : 18-23. Le slider va de 0 à 51.
        Bitrate  Débit fixe en Mb/s. Le minimum recommandé est calculé automatiquement
                 en fonction de la résolution et du chroma choisis.

    Sous-échantillonnage chroma :
        4:2:0    Standard (fichier plus petit)
        4:2:2    Meilleure qualité couleur
        4:4:4    Qualité maximale (fichier volumineux)

        Note : le bitrate et le chroma sont couplés. Modifier le chroma ajuste le
        bitrate minimum recommandé. Entrer un bitrate élevé peut ajuster
        automatiquement le chroma au niveau maximum supporté.

    Résolution de sortie :
        Original   Résolution source conservée
        1080p      1920x1080
        720p       1280x720
        576p       720x576 (PAL)
        480p       720x480 (NTSC)

        Note : les dimensions impaires sont automatiquement ajustées aux nombres pairs
        (exigence de x264/x265) via un filtre de padding.

=============================================================================================
                             16.  WORKFLOW RECOMMANDÉ
=============================================================================================

    1. Ouvrez votre source (glisser-déposer ou champ source).

    2. Naviguez jusqu’à une image représentative (ni trop sombre, ni trop claire).

    3. Activez les filtres un par un dans l’ordre du pipeline :

       a) Crop       — Supprimez les bordures noires ou ajustez le cadrage.

       b) Denoise    — Supprimez les poussières et les rayures. Le preset "standard"
                       convient dans la plupart des cas. Augmentez strength si les
                       défauts persistent.

       c) Degrain    — Commencez par le preset "Grain moyen", puis augmentez thSAD
                       si le grain persiste. Attention : des valeurs trop élevées
                       produisent un effet "plastique".

       d) Luma       — Ajustez la luminosité et le contraste. GammaY est souvent le
                       paramètre le plus utile pour éclaircir les films anciens.

       e) GamMac     — Corrigez les dérives de couleur (jaunissement, dominante
                       bleue, rouge, etc.).

       f) Sharpen    — Renforcez la netteté en dernière étape. Le mode "edge"
                       préserve mieux les zones lisses et évite d’amplifier le
                       grain résiduel.

    4. Utilisez les presets disponibles sur certains filtres pour un réglage rapide, puis affinez.

    5. Vérifiez le résultat en parcourant plusieurs passages du film.

    6. Sauvegardez votre configuration en preset dans le menu Settings pour pouvoir
       la réutiliser facilement sur d’autres clips du même film ou d’autres films similaires.

    7. Si vous avez plusieurs clips : chargez-les tous, puis appliquez les presets adaptés.

    8. Ouvrez le panneau Record, choisissez les paramètres d’encodage.
       Chaque film peut avoir un preset d’encodage différent, ce qui permet d’organiser
       facilement les rendus.
       Chaque film peut ou non être inclus dans un batch d’encodage.
       Lancez le rendu !

    Conseil : activez le mode demi-résolution (1/2) pendant le réglage des paramètres
    pour accélérer la prévisualisation, puis désactivez-le pour vérifier le résultat
    en pleine résolution avant l’encodage.


=============================================================================================
                           17.  ASTUCES ET RACCOURCIS
=============================================================================================

    - La molette de la souris sur les champs numériques ajuste la valeur par pas.
    - Le mode demi-résolution (bouton 1/2) accélère considérablement la prévisualisation.
    - Tous les paramètres sont sauvegardés automatiquement dans ScriptUser.avs à chaque
      modification.
    - La position, la taille de la fenêtre et l’état des panneaux sont mémorisés entre
      les sessions.
    - Le panneau Record est restauré à l’ouverture s’il était ouvert à la fermeture.
    - Les infobulles (au survol) sur chaque paramètre affichent les plages et des conseils.
    - Le glisser-déposer de plusieurs fichiers crée automatiquement un clip par fichier.
    - Les sliders se positionnent directement au clic (pas besoin de glisser).
    - La visite guidée (menu) permet de découvrir les bases de l’interface pas à pas.
    - Les erreurs AviSynth affichent automatiquement la ligne de script fautive.
    - Le bouton "Copier" dans la fenêtre d’erreur copie le message et les détails.
